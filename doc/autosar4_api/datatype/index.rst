DataType
========

.. table::
   :align: left

   +--------------------+-------------------------------------------+
   | Module             | autosar.datatype                          |
   +--------------------+-------------------------------------------+

The datatype module contains classes that are used to create AUTOSAR data types.

The Python autosar package primarilty supports :ref:`implementation data types <ar4_datatype_ImplementationDataType>` and has some support for 
:ref:`application data types <ar4_datatype_ApplicationDataType>`.
